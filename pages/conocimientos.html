Mis Conocimientos

Como desarrollador frontend en formación, estoy dedicando tiempo y esfuerzo para comprender los principios fundamentales de la creación de interfaces de usuario atractivas e interactivas.
Mi viaje incluye aprender a diseñar prototipos y realizar la implementación de sitios web y aplicaciones web responsivas.
Estoy enfocado en adquirir conocimientos prácticos sobre los principios de UX/UI, que son cruciales para desarrollar soluciones técnicamente eficientes y visualmente atractivas.

Aunque aún estoy en las etapas iniciales de mi carrera, me esfuerzo por adoptar un enfoque ágil para el desarrollo y valoro la importancia de la colaboración en equipo.
Estoy explorando activamente nuevas tecnologías y tendencias en la industria del desarrollo web, con un compromiso firme hacia el aprendizaje continuo para mantenerme actualizado con las últimas herramientas y prácticas.
Este enfoque asegura que estoy preparado para contribuir a proyectos innovadores y escalables en el futuro.

Tecnologías de frontend que estoy aprendiendo:

  • HTML5
  • CSS y frameworks como Tailwind
  • JavaScript y librerias como React
  • Git
  • Diseño responsivo (Responsive Design)
  • Accesibilidad web
